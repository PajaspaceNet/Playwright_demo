# Playwright POC

Jednoduchá ukázka automatizovaného testu v Pythonu pomocí Playwright.

## Jak spustit

1. Nainstaluj Playwright:
```bash
pip install -r requirements.txt
playwright install
```

2. Spusť test:
```bash
python run_tests.py
```

##Je tu nekolik moznosti spustit jednoduche testy 

